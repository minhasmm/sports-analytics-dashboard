## Prerequisites

### Required Software

- Docker & Docker Compose (v20.10+)
- Python 3.11+ (for local development)
- Node.js 18+ (for local development)
- Git

### System Requirements

- **CPU:** Minimum 2 cores
- **RAM:** Minimum 4GB
- **Storage:** 10GB free space
- **OS:** Linux, macOS, or Windows with WSL2
